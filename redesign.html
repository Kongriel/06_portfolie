Arbejd i Trello

Organiseret arbejde

Vi benyttet Trello som et centralt værktøj til at planlægge og organisere vores arbejde. Vi har anvendt SCRUM-elementer, såsom opgaveopdeling i sprints og daglige stand-up møder, for at sikre et struktureret samarbejde i vores team. Trello har gjort det nemt at visualisere og prioritere opgaver, mens SCRUM-principperne har styrket vores samarbejde og leveringen af redesignet af virksomhedens website.

Ideboard I XD

idedeling er vejen frem

I vores projekt brugte vi Adobe XD til at oprette et stort samlet ideboard til planlægning af designet til vores webside. Vi kunne nemt samle og organisere vores ideer, oprette wireframes og interaktive prototyper, hvilket hjalp os med at visualisere og kommunikere vores designkoncepter på en effektiv måde. XD's kollaborative funktioner gjorde det også nemt for vores team at arbejde sammen om designprocessen og give feedback til hinanden.

Samarbejde via GitHub

Netlify, Git og GitHub

I vores projekt udnyttede vi værktøjer som Netlify, Git og GitHub til at understøtte vores webudviklingsproces. Netlify gjorde det nemt for os at deployere og hoste vores website, mens Git og GitHub hjalp os med at samarbejde. Vi kunne effektivt håndtere ændringer, dele kode og løse konflikter, hvilket bidrog til en mere struktureret og effektiv arbejdsproces.
